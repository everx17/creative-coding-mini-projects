# ASMRnews (Front-end Website Prototype)

Multi-page front-end website prototype built in Year 1 using vanilla
HTML/CSS/JavaScript. Focused on responsive UI, interactive components, and a
day/night theme toggle.

Demo video: <paste-your-demo-video-link-here>

## Highlights
- Day/Night theme toggle (stylesheet swap)
- Responsive layout + consistent header/navigation across pages
- Newsletter subscription page with client-side validation + modal overlays
- Mock merch store: category tabs, add-to-cart, and checkout simulation
  (front-end only)
- Interactive page with audio samples + simple poll UI
- Studies/Socials pages with content switching patterns (tabs/slider-style UI)
  and URL parameters for deep links

## Run locally
This is a static site. From the project root, start a local server:

python -m http.server 8000

Then open:
http://localhost:8000
